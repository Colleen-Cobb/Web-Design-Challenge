Summary: Latitude vs. X


The purpose of this project was to analyze various weather changes as you move from one specturm of the equator to the other including cloudiness, temperature, wind speed, and humidity. Over 500 citites from across the world were analyzed to uncover these changes and the results are displayed in various scatter plots. Data from the OpenWeatherMap API was utilized to gather data in order to execute the data visualizations.


Matplotlib was used to create the visualizations between latitude and the four variable mentioned above.
